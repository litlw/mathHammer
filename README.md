# mathHammer

A python script to test all the math of your favourite tabletop strategy game!

### What does it do? 

This script is intended to do a couple of things, largely relying on the output of NewRecruit's stat cards. The functionality is pretty simple. The script opens the HTML export from NewRecruit, reads through each "card" object, scrapes the unit stats, and puts them into a big list. 

You can load multiple army lists into this script, such that you can run simulations of battle from one unit against another. 

It's far from perfect, but it does a pretty good job.
